# ManaSarukulu

మన ఇంటి డిజిటల్ సరుకుల పుస్తకం

ManaSarukulu is a smart grocery planning web app built for Telugu families.  
It helps families manage grocery shopping, monthly budgets, shopping history, and expense tracking in a simple and mobile-friendly way.

## Features

- Grocery item management
- Telugu + English grocery names
- Budget tracking
- Shopping history
- Reports and analytics
- Mobile-first responsive design
- localStorage persistence
- Modern dashboard UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- Recharts
- Lucide React Icons
- localStorage
